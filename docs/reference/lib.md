# Public library (`sparkth.lib`)

`sparkth/lib/` is the curated, stable API that application code **and plugins** import
from, instead of reaching into `sparkth.core.*` (or `sparkth.llm.*` / `sparkth.rag.*`)
directly (see [#379](https://github.com/edly-io/sparkth/issues/379)). Everything below is
generated from the module docstrings.

The plugin authoring surface and the permissions API have their own pages:
[Plugin authoring](plugins.md) and [Permissions](permissions.md).
